Proyectos

Proyecto Pokédex

En este proyecto de Pokedex construido con React y Pokeapi, el enfoque está en proporcionar a los usuarios una interfaz simple pero atractiva para navegar y buscar información sobre las criaturas de Pokemon. Las características clave de la Pokedex incluyen una barra de búsqueda, una lista de criaturas de Pokemon, una vista detallada de una criatura de Pokemon seleccionada, paginación y un diseño sensible.

La barra de búsqueda permite a los usuarios buscar criaturas de Pokemon por su nombre o parte del mismo, proporcionando una manera rápida y conveniente de encontrar la información deseada. La lista de criaturas de Pokemon muestra el nombre e imagen en miniatura de cada criatura que coincide con la búsqueda. Los usuarios pueden desplazarse por la lista y seleccionar un Pokemon para ver más detalles.

La vista detallada de una criatura de Pokemon seleccionada proporciona varias informaciones como su nombre, tipo, estadísticas, habilidades y movimientos. Los usuarios también pueden ver una imagen más grande del Pokemon y escuchar su sonido de grito, lo que hace la experiencia más interactiva y atractiva.

Proyecto BMP

El proyecto BMP ("Blazor Movie Portfolio") es un proyecto sencillo que se encarga de consultar la información relacionada sobre una película que se desee buscar y la aplicación mostrara una imagen relacionada a la película, si esta existe en la Open Movie Data Base. Al mismo tiempo se encarga de ir almacenando en cache un pequeño historial de las películas que se han ido buscando.

Lo que destaca de este pequeño proyecto es el uso de Blazor. Es un framework de desarrollo de aplicaciones web muy similar a React, basado en componentes; pero con la gran diferencia que el lenguaje de programación principal es C#. Entonces, gracias a esta herramienta podemos utilizar casi la totalidad de todas las herramientas que existen en .NET Core, dependiendo del tipo de proyecto que se cree en Blazor. Para una mayor compatibilidad con .Net Core se recomienda realizar una aplicación de Blazor Server. Blazor es compartible con la mayoría de servidores que están en el mercado y además, permite ejecutarse sobre Linux si es necesario.
